## 🚀 Быстрый старт

### Если нужно сделать вёрстку по макету:

1. Откройте `File_md/verstka_figma_md.md`
2. Следуйте алгоритму из 5 шагов
3. Используйте чек-лист для проверки

### Если нужно понять проект:

1. Откройте `File_md/README.md`
2. Перейдите в `project_status_docs/PROJECT_STATUS.md`
3. Изучите `Git_docs/Git.md`

### Если нужно протестировать:

1. Откройте `File_md/test_docs/test_auto_resize_guide.md`
2. Следуйте тестовым сценариям
