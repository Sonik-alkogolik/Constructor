## 🐛 Известные ограничения

### Image to Code
1. **Сложные сетки** - могут распознаваться неточно
2. **Перекрывающиеся элементы** - могут объединяться
3. **Время анализа** - 10-20 секунд для 1920×1080
4. **Точность OCR** - зависит от качества изображения

### Автовёрстка
1. **Глубокая вложенность** - может требовать ручной корректировки
2. **Абсолютное позиционирование** - не поддерживается
3. **Сложные grid шаблоны** - упрощаются до 1fr

## 💡 Рекомендации по использованию

### Для лучших результатов Image to Code:
1. Экспортируйте макеты в PNG высокого качества
2. Убедитесь в читаемости текста
3. Используйте простые вертикальные структуры
4. Избегайте наложений элементов

### Для автовёрстки:
1. Используйте семантические классы
2. Избегайте абсолютного позиционирования
3. Группируйте элементы логически
4. Тестируйте на всех брейкпоинтах
