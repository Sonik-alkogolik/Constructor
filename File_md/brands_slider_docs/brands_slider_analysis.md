## 🏗️ HTML Структура

```html
<div class="brands-slider">
  <div class="brand-card">
    <div class="brand-logo">
      <img src="logo.png" alt="BRAND">
    </div>
    <div class="brand-price">
      <span class="price-label">Стоимость:</span>
      <span class="price-value">ОТ 15 000 ₽</span>
    </div>
    <div class="brand-divider"></div>
    <div class="brand-stats">

      <span class="stats-value">124 АВТО</span>
    </div>
    <button class="brand-button">ЗАКАЗАТЬ</button>
  </div>
  <!-- ... ещё 5 карточек -->
</div>
```

### Семантика
- **Контейнер:** `<div class="brands-slider">` - flex-контейнер для горизонтальной прокрутки
- **Карточка:** `<div class="brand-card">` - независимый элемент
- **Логотип:** `<img>` с alt для доступности
- **Текст:** `<span>` с семантическими классами (label/value)
- **Кнопка:** `<button>` для интерактивности

## 📱 Адаптивность

### Tablet (768px)
```css
@media (max-width: 768px) {
  .brand-card {
    flex: 0 0 160px;  /* Уменьшаем ширину */
  }
}
```

### Mobile (375px)
```css
@media (max-width: 375px) {
  .brand-card {
    flex: 0 0 140px;  /* Ещё меньше */
  }
  
  .price-value {
    font-size: 14px;  /* Уменьшаем шрифты */
  }
}
```

### Принципы адаптивности
1. **Уменьшение ширины карточек:** 200px → 160px → 140px
2. **Уменьшение отступов:** 16px → 12px → 10px
3. **Уменьшение шрифтов:** для экономии места
4. **Сохранение пропорций:** все элементы масштабируются равномерно

## 🎯 Ключевые решения

### 1. Горизонтальная прокрутка
**Проблема:** 6 карточек не помещаются на экране  
**Решение:** `overflow-x: auto` с `flex` контейнером

### 2. Фиксированная ширина
**Проблема:** Карточки сжимаются неравномерно  
**Решение:** `flex: 0 0 200px` - фиксированный размер

### 3. Единый стиль кнопок
**Проблема:** Кнопки разной ширины  
**Решение:** `width: 100%` внутри карточки

### 4. Адаптивные изображения
**Проблема:** Логотипы разных размеров  
**Решение:** `object-fit: contain` + фиксированная высота
